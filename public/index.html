🔄 سير العمل الأساسي
  1. استلام الخامات: أدخل اسم الخامة + المورد + باتش المورد + الكمية. سيُنشأ رقم RM-XXXXX تلقائياً.
  2. مخزون الخامات: يعرض الرصيد الحالي = إجمالي الاستلام - إجمالي الصرف.
  3. صرف الخامات: أدخل الكمية المصروفة + باتش المنتج النهائي (FG-XXXXX) لربط الصرف بالإنتاج. رقم الصرف DSP-XXXXX يُنشأ تلقائياً.
  4. الإنتاج: أنشئ دُفعة إنتاج (FG-XXXXX). ثم أضف عملاء لكل باتش بالضغط على "عملاء" مع الكميات وأرقام الفواتير.
  5. التتبع الأمامي: ابحث بـ RM-XXXXX أو اسم الخامة ← يعرض الصرف والمنتج والعملاء.
  6. التتبع الخلفي: ابحث بـ FG-XXXXX أو اسم المنتج أو اسم العميل ← يعرض الخامات المستخدمة.
📋 أرقام الباتشات
  • RM-XXXXX — استلام الخامات
  • DSP-XXXXX — صرف الخامات
  • PKG-XXXXX — استلام التعبئة
  • PKGD-XXXXX — صرف التعبئة
  • FG-XXXXX — المنتج النهائي (Finished Goods)
⚠️ تنبيهات الصلاحية

لوحة التحكم تعرض تلقائياً كل المواد والمنتجات التي ستنتهي صلاحيتها خلال 30 يوم.
