📖 دليل المستخدم / User Manual

Batch Tracker Pro v2 – دليل الاستخدام
🔐 تسجيل الدخول

استخدم اسم المستخدم admin وكلمة المرور admin للدخول الأول.

📥 استلام المواد الخام

اذهب إلى استلام خام، أدخل اسم المادة (من القائمة أو اكتب يدوياً)، الكمية، المورد، ورقم اللوت. سيتم توليد رقم دفعة تلقائياً بصيغة RM-XXXXX.

📤 صرف المواد الخام

اذهب إلى صرف خام، أدخل المادة والكمية والغرض. رقم الدفعة سيكون بصيغة DSP-XXXXX.

📦 مخزون المواد الخام

يعرض الرصيد الحالي لكل مادة (الوارد - المصروف). الحالة تظهر ⚠️ منخفض عند الوصول للحد الأدنى.

📥 استلام مواد التعبئة

نفس آلية المواد الخام لكن لمواد التعبئة. رقم الدفعة بصيغة PKG-XXXXX.

📤 صرف مواد التعبئة

رقم الدفعة بصيغة PKGD-XXXXX.

🏭 سجلات الإنتاج

أدخل اسم المنتج النهائي، العميل (شركة أو أفراد)، الكمية، وتواريخ الإنتاج والانتهاء. رقم الدفعة بصيغة FG-XXXXX.

🔍 تتبع الدفعة

ابحث برقم الدفعة أو اسم المادة لرؤية جميع السجلات المرتبطة بها.

⚙️ إدارة المواد

أضف مواد جديدة لقائمة المواد الخام ومواد التعبئة، وحدد الحد الأدنى للتنبيه عند انخفاض المخزون.

📋 التقارير

اعرض ملخص المواد الخام، التعبئة، أو الإنتاج. يمكن طباعتها أو تصديرها كـ CSV.

أرقام الدفعات / Batch Number Format
  • RM-XXXXX – استلام مواد خام / Raw Material Receipt
  • DSP-XXXXX – صرف مواد خام / Raw Material Dispatch
  • PKG-XXXXX – استلام مواد تعبئة / Packaging Receipt
  • PKGD-XXXXX – صرف مواد تعبئة / Packaging Dispatch
  • FG-XXXXX – إنتاج / Finished Goods Production
